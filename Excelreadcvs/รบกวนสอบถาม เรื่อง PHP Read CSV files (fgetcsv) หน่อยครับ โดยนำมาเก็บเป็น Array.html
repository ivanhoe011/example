รบกวนสอบถาม เรื่อง PHP Read CSV files (fgetcsv) หน่อยครับ โดยนำมาเก็บเป็น Array

 
เริ่มหัวข้อใหม่

Guest


รบกวนสอบถามหน่อยครับ คือว่าผมมีงานที่จำเป็นจะต้องเก็บข้อมูลด้วย CSV น่ะครับ โดยข้อมูลของผมจะเก็บโดยการตั้งชื่อ เป็น วันที่.csv เช่น 01_01_2011.csv น่ะครับ ซึ่งจะมีการเก็บข้อมูลทุกวันที่มีการเปลี่ยนแปลงข้อมูลครับ ซึ่งข้อมูลก็จะถูกเก็บไปเรื่อย ๆ ครับ แต่ตอนนี้ผมกำลังจะนำข้อมูลเหล่านั้นมาทำ Report ครับ ซึ่งผมจะมีหน้าสำหรับให้กรอก ว่าต้องการจะดูข้อมูลในช่วงวันที่เท่าไร ถึง เท่าไรน่ะครับ แล้วก็มีปุ่ม GenReport น่ะครับ พอกดก็จะโชว์รายการข้อมูลตามที่เรากรอกไปน่ะครับ

ซึ่งผมได้เห็นโค๊ดในเว็บนี้ เป็นการนำข้อมูลมาเก็บไว้ในรูปของ Array แล้วแสดงผล มันก็ใกล้กับความต้องการของผมครับ แต่มันเป็นการแสดงข้อมูลเพียงแค่ไฟล์เดียวน่ะครับ คือผมต้องการจะให้แสดงได้ตามวันที่ ที่เรากรอกไปน่ะครับ เช่น ผมกรอก วันที่ 01-01-2011 ถึง 03-01-2001 ก็จะต้องแสดงข้อมูลของวันที่ 1-3 มาให้น่ะครับ รบกวนช่วยบอกหน่อยครับ ว่าผมต้องเขียนโค้ดต่ออย่างไงดีครับ

Code (PHP)
<html>
<head>
<title>ThaiCreate.Com PHP & Read CSV</title>
</head>
<body>
<?
$objCSV = fopen("customer.csv", "r");
?>
<table width="600" border="1">
  <tr>
    <th width="91"> <div align="center">CustomerID </div></th>
    <th width="98"> <div align="center">Name </div></th>
    <th width="198"> <div align="center">Email </div></th>
    <th width="97"> <div align="center">CountryCode </div></th>
    <th width="59"> <div align="center">Budget </div></th>
    <th width="71"> <div align="center">Used </div></th>
  </tr>
<?
while (($objArr = fgetcsv($objCSV, 1000, ",")) !== FALSE) {
?>
  <tr>
    <td><div align="center"><?=$objArr[0];?></div></td>
    <td><?=$objArr[1];?></td>
    <td><?=$objArr[2];?></td>
    <td><div align="center"><?=$objArr[3];?></div></td>
    <td align="right"><?=$objArr[4];?></td>
    <td align="right"><?=$objArr[5];?></td>
  </tr>
<?
}
fclose($objCSV);
?>
</table>
</body>
</html>

			if ($objFopen) {
				while (!feof($objFopen)) {
					$file = fgets($objFopen, 4096);
					$arr[] = $file;
				}
				fclose($objFopen);
			}

	if($arr[$i] != "")

	}
}
?>

ขอบคุณมาก ๆ เลยครับ คุณ mr.win ที่ช่วยอธิบาย และทำตัวอย่างให้ดูครับ แต่ของผมจะเป็นการค้นหาข้อมูลจากไฟล์ CSV น่ะครับ
คือจะมีฟอร์มให้กรอกช่วงวันที่ที่ต้องการจะดูข้อมูล แล้วก็แสดงผลออกมาน่ะครับ การทำงานจะคล้าย การค้นหาจากฐานข้อมูลที่เป็น mysql ครับ แต่ของผมจำเป็นต้องเก็บไฟล์เป็น CSV น่ะครับ ลำดับการทำงานของผมจะเป็น ดังนี้

1. กรอกวันที่ ที่ต้องการจากฟอร์ม (เช่น กรอก วันที่ 01_01_2011 ถึง 01_03_2011)
2. เช็คว่าวันที่ตรงกับชื่อไฟล์ของ CSV อะไรบ้าง (เช็คว่ามีไฟล์ของวันที่ 1-3 หรือไหม)
3. อ่านไฟล์ CSV นั้น เก็บค่าใส่ Array (อ่านไฟล์ของวันที่ 1-3 เก็บค่าใส่ Array ไว้)
4. แสดงผลข้อมูลเป็นตารางตามรูปแบบที่กำหนด (Query แล้วแสดงผลข้อมูล)

ลักษณะการทำงานที่ผมต้องการจะประมาณนี้ครับ รบกวนคุณ mr.win แนะนำหน่อยครับ ว่าจะต้องเริ่มจากตรงไหน เพราะตามตัวอย่างที่คุณ mr.win ให้มามันจะเป็นการอ่านไฟล์ทั้งหมดเก็บใส่ array แล้ว insert ลงฐานข้อมูล mysql อ่ะครับ ซึ่งของผมไม่ใช่แบบนั้นครับ คือจะ query ข้อมูลจาก ไฟล์ CSV เลยน่ะครับ
