เกริ่นก่อน...ตั้งแต่เริ่มแรกเดิมที่ เคยทำบันทึกข้อมูลใน google calendar แต่บ้างครั้ง ข้อมูลมันไม่อัพเดท 
ช้าบ้าง ปัญหาสารพัน (ที่เกิดจากตัวผู้ใช้งาน) จึงทำให้ค้นหาพวกปฏิทินกิจกรรมต่าง ๆ จนไปเจอตัว 
fullcalendar ศึกษาอยู่พักใหญ่ นานมากกว่าจะได้ออกมาแบบต้องการ ตามการใช้งานจริง 
จึงเอาไปให้งานยานพาหนะใช้งาน ปรับแก้ข้อมูลกว่าจะตรงใจผู้ใช้ก็เล่นเอาเหนื่อยเหมือนกัน
โดยส่วนตัวผมจะศึกษาเกี่ยวกับ Code ส่วนเรื่อง html จะให้น้อง webmaster เป็นคนจัดการ 
ดังนั้นการใช้งาน css style ต่าง ๆ ผมไม่แน่ใจเลย เพราะฉนั้นไม่อาจตอบได้

เอาล่ะ ลองดูกันเลยแล้วกัน 
=> โดยการ run ไฟล์ install.php (localhost/cars/install.php)เพื่อสร้างฐานข้อมูล ตารางผู้ดูแลระบบและตารางการเก็บข้อมูล
